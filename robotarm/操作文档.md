**遥操作模拟不带摄像头**
目的：测试硬件环境，主从机械臂的连接，此命令用于启动 SO100 机器人的遥操作模式。

```plain

python lerobot/scripts/control_robot.py teleoperate  --robot-path lerobot/configs/robot/so100.yaml    --robot-overrides "~cameras"   --display-cameras 0
```

参数说明：

+ `--robot-path lerobot/configs/robot/so100.yaml`：指定机器人的配置文件路径。
+ `--robot-overrides "~cameras"`：禁用摄像头功能。
+ `--display-cameras 0`：不显示摄像头画面。

 **采集数据：**

```plain
python lerobot/scripts/control_robot.py record    --robot-path lerobot/configs/robot/so100.yaml     --fps 30     --root data     --repo-id pick/so100_test    --tags pickbottle     --episode-time-s 30     --reset-time-s 6     --num-episodes 50     --push-to-hub 0
```

此命令用于记录 SO100 机器人的操作数据集。

参数说明：

+ `--fps 30`：设置记录的帧率为每秒 30 帧。

+ `--root data`：将数据保存到 `data` 目录。

+ `--repo-id pick/so100_test`：指定数据集的标识符。

+ `--tags pickbottle`：为数据集添加标签。

+ `--episode-time-s 30`：每个记录片段的持续时间为 30 秒。

+ `--reset-time-s 6`：每次重置之间的间隔时间为 6 秒。

+ `--num-episodes 50`：总共记录 50 个片段。

+ `--push-to-hub 0`：不将数据集上传到 Hugging Face Hub。
  **回放数据：**

```plain
python lerobot/scripts/control_robot.py replay     --robot-path lerobot/configs/robot/so100.yaml     --fps 30     --root data     --repo-id pick/so100_test     --episode 0     --robot-overrides "~cameras"
```

此命令用于回放指定的数据集片段。
参数说明：

+ `--episode 0`：回放第 0 个片段。

+ `--robot-overrides "~cameras"`：禁用摄像头功能
  **查看数据：**

```plain
python lerobot/scripts/visualize_dataset.py     --repo-id  pick/so100_test    --root ./data     --local-files-only 1     --episode-index 0
```

+ **训练数据：**

```plain
set DATA_DIR=./data/   设置训练本地数据
 python lerobot/scripts/train.py   dataset_repo_id=pick/so100_test    policy=act_so100_real     env=so100_real     hydra.run.dir=outputs/train/act_so100_test     hydra.job.name=act_so100_test     device=cuda     wandb.enable=true （同步日志到wandb需要翻墙）
```

此命令用于训练控制 SO100 机器人的策略模型。

参数说明：

+ `DATA_DIR=./data/`：指定数据集所在目录。
+ `dataset_repo_id=pick/so100_test`：使用之前记录的数据集。
+ `policy=act_so100_real`：指定策略配置。
+ `env=so100_real`：指定环境配置。
+ `hydra.run.dir=outputs/train/act_so100_test`：设置训练输出目录。
+ `hydra.job.name=act_so100_test`：设置训练任务名称。
+ `device=cuda`：使用 GPU 进行训练。
+ `wandb.enable=true`：启用 Weights and Biases 进行训练监控。
  
  

 **模型评估：**

```plain

python lerobot/scripts/control_robot.py record     --robot-path lerobot/configs/robot/so100.yaml     --fps 30    --root data     --repo-id pick/eval_so100_test     --tags pickbottle     --warmup-time-s 5     --episode-time-s 300     --reset-time-s 10     --num-episodes 2     -p outputs/train/act_so100_test/checkpoints/last/pretrained_model
```

此命令用于评估训练好的策略模型在 SO100 机器人上的表现。

参数说明：

+ `--robot-path lerobot/configs/robot/so100.yaml`：指定机器人的配置文件路径。
+ `--fps 30`：设置数据采集的帧率为每秒 30 帧。
+ `--root data`：指定数据存储的根目录为 `data` 文件夹。
+ `--repo-id pick/eval_so100_test`：设置评估数据集的标识符为 `pick/eval_so100_test`。
+ `--tags pickbottle`：为评估数据集添加标签 `pickbottle`，便于分类和检索。
+ `--warmup-time-s 5`：在每个回合开始前有 5 秒的预热时间。
+ `--episode-time-s 300`：每个回合的持续时间为 300 秒（5 分钟）。
+ `--reset-time-s 10`：每个回合之间的重置时间为 10 秒。
+ `--num-episodes 2`：总共记录 2 个回合的评估数据。
+ `-p outputs/train/act_so100_test/checkpoints/last/pretrained_model`：指定使用训练好的策略模型，该模型位于 `outputs/train/act_so100_test/checkpoints/last/pretrained_model`
